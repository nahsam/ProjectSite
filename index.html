Wearable Awareness System (Senior Design Project)

Description:

The project was a wearable device that would recognize certain sounds and give haptic feedback to the user for emergency situations. This was intended for the hearing-impaired community, designed to be used offline with all recognition functionality on device. This project was cut short due to COVID in 2020 and was not completed.

Features

  • Model trained on Google CoLabs utilizing TensorFlow
  • Implemented on Teensy4.0 Micro controller
  • Integrated microphone for testing input sounds
  • Haptic feedback vibration motor
  • Backlit digital display
